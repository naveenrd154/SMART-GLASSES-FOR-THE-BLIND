# SMART-GLASSES-FOR-THE-BLIND
Developed smart wearable glasses using an ultrasonic sensor for obstacle detection. Integrated an NE555 timer to trigger a buzzer alert and powered the system with a 3.7V battery. Optimized mobility, safety, and confidence for visually impaired users.

## Features
- Real-time obstacle detection
- Audio alert using buzzer
- Portable and battery powered design

## Hardware Used
- Ultrasonic Sensor
- NE555 Timer
- Buzzer
- 3.7V Battery

## What I Learned
- Ultrasonic sensor interfacing
- Timer-based alert systems
- Practical circuit design and testing

## Why this project matters to me
This project helped me apply electronics fundamentals to build a real-world assistive solution for visually impaired users.
